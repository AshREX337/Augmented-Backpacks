**Backpacked** is a great mod by MrCrayfish but I always felt that it needed more Augments because theres never such a thing as too many.
So I took it upon myself to add some of those useful ones that you see in other Backpack mods while still keeping it balanced to MrCrayfish's style.

If you have any issue or have an idea for a new augment to add, leave a comment!
Currently only available for 1.21.1 NeoForge (I may port to others but it will take a while)
